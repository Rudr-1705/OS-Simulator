Cache Simulation

Configure cache size, associativity, and replacement policy. Observe hits, misses, and AMAT across access patterns.

Key Concepts

Direct Mapped
Each memory block maps to exactly one cache set. Simple but prone to conflict misses.
Set Associative
k-way: memory blocks map to a set of k cache lines. Reduces conflicts vs direct mapped.
Fully Associative
Any memory block can go into any cache line. Eliminates conflict misses but requires searching all lines.
Cold Miss
First access to a block — compulsory miss. Cannot be avoided without prefetching.
Capacity Miss
Cache is full; block evicted and re-accessed later. Reduce by increasing cache size.
AMAT
Average Memory Access Time = Hit Time + Miss Rate × Miss Penalty. Key performance metric.
