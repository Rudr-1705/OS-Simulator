Memory Management Unit

Simulate virtual to physical address translation with TLB hits and misses.

Key Concepts

MMU
Hardware device that maps virtual to physical addresses at runtime.
TLB
Translation Lookaside Buffer. A fast associative cache for page table entries.
VPN / Offset
Virtual address is split into a Virtual Page Number and an Offset.
Page Walk
The process of traversing the page table in memory when a TLB miss occurs.
